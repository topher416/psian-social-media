# Canva Image Creation Instructions

## Overview
Create **30 images** for the January 2026 batch of PsiAN social media posts.

---

## Setup

### Canva Project Settings
- **Create**: New project with multiple pages
- **Size**: 1080 x 1080 px (square - works for IG, FB, LinkedIn, Twitter)
- **Pages**: 30 total (one per post)

### Export Settings
- **Format**: PNG
- **Pages**: All pages
- **Download**: Will auto-name as `PsiAN Social Media - 1.png`, `PsiAN Social Media - 2.png`, etc.

### After Export - Rename Files
Rename to match post numbers:
```
01.png, 02.png, 03.png... 30.png
```

## Design Guidelines

### Brand Elements
- **Logo**: PsiAN logo in corner (consistent placement)
- **Colors**: Match PsiAN brand colors
- **Fonts**: Professional, readable sans-serif

### Text Overlay Rules
- **Maximum**: 10 words
- **Placement**: Center or lower third
- **Contrast**: Ensure text is readable over background
- **Size**: Large enough to read on mobile

### Generic Image Rules
- Logo placement: Bottom right corner
- Keep imagery calm, professional, hopeful
- Nature imagery works well for therapy themes
- Avoid clich√©d "head in hands" depression imagery
