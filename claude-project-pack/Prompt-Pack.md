Use these prompts inside a Claude Project after loading the other docs.

## 1. Read The Sources First

Read `Project-Brief.md` and `Source-Map.md` first.
Use PsiAN's site language as the source of truth.
Then create a social post idea that matches the organization's voice.

## 2. Generate Hooks

Create 10 hook options for a PsiAN post about:
[topic]

Constraints:
- 1 to 2 sentences each
- professional and measured
- no hype
- no emojis
- make the first line strong
- suggest a different angle for each hook

## 3. Turn A Source Into A Post

Using this PsiAN page or article:
[paste source or link]

Write:
- one hook
- one Instagram/Facebook/Threads caption
- one LinkedIn version
- one X thread opener plus 3 replies
- one Bluesky version

Rules:
- keep the voice consistent with PsiAN
- lead with the main point quickly
- use clear, readable paragraphs
- end with a next step or PsiAN reference
