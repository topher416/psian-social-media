# PsiAN Claude Project Kit

This folder is meant to be added to a Claude Project as the working reference for PsiAN social media drafting.

What it gives Claude:
- Background on PsiAN and the assignment
- A source map of the PsiAN site to read first
- Voice, hook, and social media best practices
- Prompt templates for generating drafts, hooks, and variations

How to use it:
1. Upload this folder to the Claude Project.
2. Tell Claude to read `Project-Brief.md` first.
3. Use `Source-Map.md` before drafting anything new.
4. Use `Prompt-Pack.md` to generate posts, hooks, and platform variants.

The goal is simple:
- Keep PsiAN visible with multiple posts per week
- Pull from PsiAN's existing publications, positions, and site content
- Make drafting easier for someone who is new to social media
- Keep the tone professional, measured, educational, and clear
